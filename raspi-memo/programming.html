6.3. Ruby¶

使用Raspbian系统中的ruby包安装：

$ sudo aptitude install ruby ruby-dev
$ sudo gem install pi_piper

如果使用rvm安装，要另外为root用户安装一份，以为对GPIO编程需要root权限。
